# VET INFO MANAGEMENT SYSTEM

This is a veterinarian Doctor Information system with the inventory part. The system has been developed in OOP PHP, MySQLi with other web standards.

## System Modules

- Animals Module
- Clients Module
- Appointment Module
- Medical Records Module
- Inventory Module
- Expenses Module
- Payroll Module
- Messaging Module
- Users Module
- and a few more.
